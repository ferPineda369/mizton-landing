# Assets Folder - KIMEN Token Landing Page

## Required Images

Place the following images in this folder:

### 1. portada-libro.jpg
- **Description**: Book cover for "Un Bello Mundo por Extrañar"
- **Recommended size**: 800x1200px (portrait)
- **Format**: JPG or PNG
- **Usage**: Displayed in "EL LIBRO" section

## Image Guidelines

- Use high-quality images (at least 1200px width)
- Optimize images for web (compress without losing quality)
- Use descriptive filenames
- Supported formats: JPG, PNG, WebP

## Placeholder

Until you add the actual book cover, a placeholder will be shown. Replace `portada-libro.jpg` with your actual book cover image.

You can create a temporary placeholder by:
1. Creating a 800x1200px image with the book title
2. Saving it as `portada-libro.jpg` in this folder
3. Replacing it later with the final design
